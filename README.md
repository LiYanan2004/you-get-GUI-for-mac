# you-get-GUI-for-mac

This is a GUI for you-get script which offers a visual experience.

![](/imgs/screenshot.png)

## Reqiurements

* you-get
* Python 3.8+
* macOS Ventura
* (Optional) ffmpeg

## How to use

1. Open App
2. Fill in your media link
3. Change settings if you need
4. Hit download button
